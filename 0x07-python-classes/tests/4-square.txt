The Square Module:

    >>> Square = __import__("4-square").Square

Correct Usage:

    >>> square = Square()
    >>> print("{:d}".format(square.size))
    0
    >>> print('{}'.format(square.area()))
    0

    >>> s = Square(5)
    >>> print("{:d}".format(s.size))
    5
    >>> print('{}'.format(s.area()))
    25

    >>> s.size = 9
    >>> print("{:d}".format(s.size))
    9
    >>> print('{:d}'.format(s.area()))
    81

    >>> print(s.__dict__)
    {'_Square__size': 9}
